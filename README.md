# GitHub Pages Project

This project is a simple web application hosted on GitHub Pages. It includes several components:

- **ashravan.txt**: A short story inspired by Brandon Sanderson's work, exploring the fate of Ashravan after Shai restores him.
- **dilemma.json**: A JSON file describing a scenario where an autonomous vehicle must make a critical decision.
- **index.html**: The main HTML file that ties everything together.
- **styles.css**: A CSS file for styling the HTML page.
- **script.js**: A JavaScript file for adding interactivity to the page.

## How to View

To view the project, simply navigate to the GitHub Pages URL associated with this repository.
